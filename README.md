# 中康国悦站点源码重建

这是根据已部署的 Vercel 项目 `zkgy-clone` 重建的 Next.js App Router 项目。

## 本地开发

```bash
npm install
npm run dev
```

默认本地地址：

```text
http://localhost:3000
```

## 已实现路由

- `/`
- `/about`
- `/contact`
- `/faq`
- `/coming-soon`
- `/admin`
- `/api/contact`
